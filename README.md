# P2P NFT Lending Borrowing Platform by @uok825

## Deployment

Create an .env file using the .env.example file as a template.

Then run:

```bash
  yarn install
```

App is setup to deploy on goerli. To deploy and start the app

```bash
  yarn deploy
  yarn start
```
